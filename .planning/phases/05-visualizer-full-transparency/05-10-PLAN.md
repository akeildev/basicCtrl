## Troubleshooting

| Issue | Diagnosis | Fix |
|---|---|---|
| Ghost cursor not visible | Visualizer.swift not running | `ps aux | grep Visualizer`, restart Swift sidecar |
| HUD not updating | VisualizerBus socket not ready | Check `/tmp/cua-visualizer.sock` exists |
| Recording not created | ScreenRecorder.swift failed | Check console logs for VideoToolbox errors |
| Replay state mismatch | action_log.ndjson corrupted | Verify NDJSON format: `jq empty` on file |
| Timeline lag on 1000+ nodes | Canvas rendering slow | Profile with Xcode Instruments (CoreAnimation) |

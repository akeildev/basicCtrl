# Phase 03 Plan 07: Cassette Replay Engine (Wave 4)

**Implement cassette replay that runs recorded steps, detects mismatches via pHash diff, and falls through to live RaceOrchestrator on first mismatch (D-19).**

## Objective

Implement cassette replay that deterministically re-executes a recorded sequence of actions. Compare actual screenshot after each step to recorded pHash; on mismatch (indicating selector drift or UI change), stop replay and trigger live re-execution via RaceOrchestrator. This enables fast paths (cache hits skip planning entirely) and feeds into write-back for healed selectors.

Purpose: Enable cache-hit fast paths; detect when cached recipes are stale; setup for Plan 03-08 write-back.

Output: `basicctrl/cache/replay.py` with full replay + mismatch detection + 12+ unit tests.

  <files>basicctrl/cache/replay.py</files>
  <action>Create replay.py:

  1. **CassetteReplayEngine class**:
     - `__init__(cassette: Cassette, race_orchestrator, l1_cheap, session_writer, target_pid: int, bundle_id: str)`
     - `_cassette: Cassette`
     - `_race_orchestrator: RaceOrchestrator` (for fallthrough on mismatch)
     - `_l1_cheap: L1CheapDiff` (for pHash snapshot comparison)
     - `_session_writer: SessionWriter` (for replay events)
     - `_phash_threshold: int = 8` (D-23 — max Hamming distance for pHash match)

  2. **async def replay() -> tuple[bool, Optional[int], List[dict]]**:
     - Returns: (success: bool, first_mismatch_step_idx: Optional[int], replay_events: List[dict])
     - Core loop:
       ```
       for step_idx, step in enumerate(cassette.steps):
           # Replay: execute the recorded action
           outcome = await race_orchestrator.execute(
               bundle_id=bundle_id,
               target_key=step.action_canonical.target,
               ...
           )
           
           # Verify: take screenshot, compute pHash
           current_snapshot = await l1_cheap.snapshot(target_pid, bundle_id)
           current_phash = current_snapshot["phash"]
           
           # Match: compare pHash
           hamming = hamming_distance(current_phash, step.screenshot_phash)
           if hamming > phash_threshold:
               # Mismatch detected
               emit event: {"event": "cassette_mismatch", "step_idx": step_idx, "hamming": hamming, ...}
               return (False, step_idx, replay_events)
           
           # Match: continue to next step
           emit event: {"event": "cassette_step_replay_ok", "step_idx": step_idx, ...}
       
       # All steps matched
       return (True, None, replay_events)
       ```

  3. **Helper: hamming_distance(h1: str, h2: str) -> int**:
     - Count differing bits between two hex pHash strings
     - Convert each hex digit to 4-bit binary, count differing bits
     - Return total count (0-64 for 64-bit hashes)

  4. **Error handling:**
     - If screenshot capture fails: treat as mismatch, fallthrough to live
     - If race_orchestrator.execute raises: log and treat as mismatch
     - If pHash is None: fallthrough to live (no baseline)

  Include module docstring explaining replay strategy and pHash threshold.

  ```python
  import hashlib
  from typing import Literal

  def compute_cache_key(
      bundle_id: str,
      role_path: str,
      instruction: str,
  ) -> str:
      """
      Compute SHA-256 cache key per D-17.
      
      Args:
          bundle_id: e.g., "com.apple.iWork.Pages"
          role_path: e.g., "AXApplication > AXWindow > AXButton"
          instruction: e.g., "click the Format button"
      
      Returns:
          SHA-256 hex digest (64 chars)
      """
      key_bytes = f"{bundle_id}:{role_path}:{instruction}".encode('utf-8')
      return hashlib.sha256(key_bytes).hexdigest()
  ```

  <files>basicctrl/cache/agent_cache.py</files>
  <action>Create agent_cache.py:

  1. **AgentCache class**:
     - `__init__(session_dir: Path)` — session_dir = ~/.cua/sessions/<id>/
     - `_cassettes_dir: Path = session_dir / "cassettes"` (create if not exists)
     - `_lock: asyncio.Lock` (per Phase 2 pattern for thread-safe access)
     - `_cache: dict[str, Cassette]` (in-memory cache for fast lookups)

  2. **AgentCache.get(bundle_id: str, role_path: str, instruction: str) -> Optional[Cassette]**:
     - Compute cache_key via compute_cache_key()
     - Check _cache (in-memory hit)
     - If miss: check disk at `_cassettes_dir / f"{cache_key}.jsonl"`
     - If disk hit: deserialize via Cassette.from_ndjson(), populate _cache, return
     - If disk miss: return None

  3. **AgentCache.put(cassette: Cassette) -> None**:
     - Lock _cache
     - Serialize cassette via to_ndjson()
     - Write to `_cassettes_dir / f"{cassette.cache_key}.jsonl"` (atomic: write .tmp, fsync, rename)
     - Update _cache[cache_key] = cassette
     - Unlock

  4. **AgentCache.clear(cache_key: str) -> None**:
     - Delete disk file
     - Remove from _cache

  5. **Error handling:**
     - If cassette file corrupted (JSON parse error): log warning, return None (treat as cache miss)
     - If disk write fails: log error (don't crash), continue (degraded but functional)

  Include module docstring explaining AgentCache contract and disk layout.

# Phase 03 Plan 08: Cassette Write-Back with Stable-Tier Gate (Wave 4)

**Implement write-back that atomically updates cassettes with healed selectors; enforce stable-tier gate (AX locators only, never vision/coordinate); implement stream caching wrapper (CACHE-03).**

## Objective

Implement write-back logic that atomically updates cassettes with healed selectors, but only for stable tiers (AXIdentifier, AXLabel, AXTitle, AXRoleDescription). Vision-based and coordinate-based heals stay session-only (never written back to canonical cassette per D-20, P23). Also implement transparent stream caching wrapper for streaming results.

Purpose: Enable self-healing cassettes that improve over time (healed selectors written back); enforce stability gate to prevent pixel-drift heals from polluting the cache; support streaming result caching for chunked outputs.

Output: `basicctrl/cache/writeback.py` with WriteBack class + stream wrapping + 15+ unit tests.

  <files>basicctrl/cache/writeback.py</files>
  <action>Create writeback.py:

  1. **WriteBack class**:
     - `__init__(cassettes_dir: Path, session_writer: SessionWriter)`
     - `_cassettes_dir: Path` — where cassettes live
     - `_session_writer: SessionWriter` — for write-back events
     - `_lock: asyncio.Lock` — protect cassette file writes (per Phase 2 pattern)

  2. **async def heal(heal_event: HealEvent, cassette_path: Path, step_idx: int) -> bool**:
     - D-20: Gate on stable-tier: `if not heal_event.is_stable_tier(): return False`
     - Load cassette from cassette_path via Cassette.from_ndjson()
     - Find step at step_idx
     - Replace step's target with healed selector: `step.target.locator = heal_event.new_locator`
     - Append heal_event to step.healed_selectors list
     - Write cassette atomically:
       a. Serialize via Cassette.to_ndjson()
       b. Write to `{cassette_path}.tmp`
       c. os.fsync to ensure durability
       d. os.rename to {cassette_path} (atomic on POSIX)
     - Emit write-back event: `{"event": "cassette_writeback", "cache_key": ..., "step_idx": step_idx, "heal_tier": heal_event.locator_tier, ...}`
     - Return True on success, False if stable-tier gated or error

  3. **Error handling:**
     - If cassette file missing: log warning, return False (cascade to user escalation)
     - If cassette parse error: log error, return False
     - If atomic rename fails: log error, return False (degraded but functional)

  4. **StreamCache wrapper** (for CACHE-03):
     - `class StreamCache`:
       - `__init__(stream_name: str, agent_cache: AgentCache)`
       - `async def __aiter__()` wrapper around an async generator
       - Caches each yielded chunk transparently
       - On re-run, replays cached chunks
     - Minimal implementation: just track chunks per stream; Phase 4+ refines

  Include module docstring explaining write-back strategy and stable-tier gate.
